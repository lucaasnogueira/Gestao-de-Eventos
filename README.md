# 🎟️ TicketFlow — Event Management System

A robust event and ticket management platform built with **.NET and C#**, designed to handle event creation, ticket sales, and attendee management workflows.

---

## ✨ Features

- 🎪 **Event Management** — Create, update, and cancel events with full lifecycle control
- 🎫 **Ticket Sales** — Handle ticket issuance, availability, and purchase logic
- 👥 **Attendee Management** — Register and track attendees per event
- 📊 **Event Dashboard** — Overview of active events and ticket status
- 🔐 **Authentication** — Secure access control for organizers and attendees

## 🚀 Getting Started

### Prerequisites

- [.NET SDK 8.0+](https://dotnet.microsoft.com/download) installed
- Visual Studio 2022 or VS Code with C# extension

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/lucaasnogueira/Gestao-de-Eventos.git
cd Gestao-de-Eventos

# 2. Navigate to the project folder
cd TicketFlow

# 3. Restore dependencies
dotnet restore

# 4. Run the application
dotnet run
```

The API will be available at:
- **API:** `http://localhost:5000`
